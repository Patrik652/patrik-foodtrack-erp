# FoodTrack GitHub Demo Script

Krátky GitHub-facing walkthrough na `60-90` sekúnd:

1. Otvor README a ukáž CI badge, Mermaid architektúru a `Tech Highlights`.
2. Scrollni na `Swagger Preview` a povedz, že API je postavené na ASP.NET Core 8, EF Core 8 a SQLite.
3. Otvor `Quick Demo` alebo Postman collection a ukáž flow:
   - operator login cez badge/PIN
   - receive batch
   - dispatch stock cez FIFO
   - recall batch
   - dashboard alerts
4. Zvýrazni, že solution obsahuje backend, testy, Docker a reálny .NET MAUI shell.
5. Záver:
   - `36+ automated tests`
   - Dockerized API
   - GitHub Actions build + test pipeline

- Ak nahrávaš nový vlastný screencast, ponechaj tento súbor ako storyboard a prepíš iba video asset.
